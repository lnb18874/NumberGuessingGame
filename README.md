# Number Guessing Game

A simple Java console-based number guessing game.

https://roadmap.sh/projects/number-guessing-game

## How to Play

The program picks a random number between 1 and 100. You have to guess it.

## Difficulty Levels

| Level  | Chances |
|--------|---------|
| Easy   | 10      |
| Medium | 5       |
| Hard   | 3       |

## Running

```bash
javac src/Main.java
java -cp src Main
```
